Para regresar los cambios anteriores solo se coloca git checkout -- <"file">

Para mostrar diferencias de cambios se coloca git diff <"file">

Ingnorar archivos carpetas para que no se agrega un arhivo en el area de trabajo

.gitingnore y adentro se coloca el nombre de los archivos o carpeta

Para crear mini versiones, primero debes tener todo guardado y colocar git branch nombredeversion sin espacios y sin comillas

Para mostrar los nombres de versiones se coloca git branch

Para moverse dentro del nombre de la version y trabajar desde ahi se coloca git checkout nombredeversion

Para acutalizar o subir cambios a un repositorio de guthub se debera abrir la consola de git y agregar los combaios con "git add." posteriormente agregar un commit "git commit -m mensaje" y por ultimo agregar el comando de "git push -u"

Antes de hacer el push, primero se debe ejecutar el comano "git pull origin master"
